Nhúng javascript vào trang html

        * Có 3 cách:
            - Sử dụng javascript trong thuộc tính sự kiện của phần tử html (inline)
                VD: onclick, onmouseover, onkeyUp, onkeyDown, onfocus,... 
            - Sử dụng các mã lệnh bên trong thẻ script 
            - Sử dụng file js ngoài, nhúng vào trang html thông qua thẻ script với thuộc tính src
