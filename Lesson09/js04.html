Các cấu trúc điều khiển trong ngôn ngữ lập trình

        * 3 loại cấu trúc điều khiển:
        1. Lựa chọn / rẽ nhánh:
            * if (4 dạng if khác nhau:)

            * switch
        2. Lặp:
            * for 
            * while 
            * do ... while 
            * foreach
        3. Lệnh nhảy:
            break;
            continue;
            return; 
            ...
