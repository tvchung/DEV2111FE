        - hiển thị hình ảnh trên html5:
            + sử dụng thẻ img:
                => thuộc tính: 
                    src -> (url) đường dẫn đến file ảnh (.jpg, .png, ...)
                    alt -> Nội dung hiển thị khi nguồn ảnh lỗi, (hover chuột)
        - url: 
            + url: tương đối 
                    +> vị trí file html hiện tại: (tên file anh.jpg)

                    +> ảnh ở bên ngoài thư mục chứa vị trí file html hiện tại 
                        ../ => ra ngoài một thư mục
