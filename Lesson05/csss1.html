Nhúng css vào trang html5

        1. style in-line: sủ dụng thuộc tính style của phần tử html5 
        2. tag style: phần tử style trên trang html 
            - các quy tắc sẽ được áp dụng cho các phần tử trên trang html 
        3. file .css (file ngoài):
            - Nhúng vào trang html thông qua thẻ link 
            - Áp dụng trên những trang có liên kết file css
