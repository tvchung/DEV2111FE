Cấu trúc lặp for 

        Cấu trúc tổng quát:

        for(init ; condition_expression; step){
            statement-block;
        }

        => có thể viết dưới 4 dạng khác nhau: 
        - đầy đủ: (như trên )
        - khuyết: 
            + khuyết init 
            + khuyết condition 
            + khuyết step
        - Khuyết toàn bộ: for (; ;){....};
        - for không thân: for(...;...;...);
